# Prose package

This directory contains the prose-facing production files.

## Files

- `canon/prose/PROSE_SYSTEM.md`: global page-level prose law. It governs how material claims must be earned on the page, including show-don't-tell blockers.
- `canon/prose/MIELLE_PROSE_STYLE_GUIDE.md`: Mielle-lens narration, first-person POV, attention order, internal rhythm and scene texture.
- `canon/prose/MIELLE_VOICE_GUIDE.md`: Mielle's spoken voice and dialogue behavior.
- `canon/prose/ANTI_AI_PROSE_EDITING_FILTER.md`: revision diagnostics, clustered failure checks and blocker categories.
- `canon/prose/CHAPTER_PROSE_AND_CHARACTER_VOICE_PASS.md`: application pass tying prose, voice, character dossiers, source, Holmes, combat and Anti-AI checks together.

## Boundaries

`canon/HOLMES_ENGINE.md` remains the authority for inference. `canon/COMBAT_RULES.md` remains the authority for combat. The prose system includes only the page-level application of those rules.

This directory does not decide canon, plot, powers, tactics, relationship status, source outcomes or continuity.
