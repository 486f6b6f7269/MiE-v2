Use this table only for required/result-bearing applications.

Rules:

- `Pressure behavior` must show what becomes shorter, rougher, stranger, more active, more avoidant or more mistaken under this exact pressure.
- `Practical competence` cannot be replaced by a voice adjective when the required result depends on a natural action lane.
- A productive consequence must change action, timing, information, object state, available choice, tactical line, relationship distance or scene outcome.
- A restrictive boundary must name the invalid result it blocks and the allowed replacement or omission.
- A correct line with no consequence should be reclassified as optional texture unless planning assigned a required result that it fails to deliver.

## Receipt validity

The receipt is invalid when:

- obligation levels are absent or silently strengthened;
- it lists only file paths, traits or voice labels;
- a required/result-bearing character has no stable/current-state synthesis;
- a required/result-bearing character has no productive consequence or restrictive boundary;
- a required/result-bearing beat fails removal without reclassification;
- required ownership is replaceable without explanation;
- practical competence required for a result is ignored;
- later-canon knowledge or maturity leaks into the current phase;
- existing ties are replaced by reaction to Mielle;
- every present character receives a compulsory beat for coverage;
- optional texture accumulates into a demonstrated repeated-function, cast-roll-call or flattened-dialogue cluster;
- a substantial revision lacks comparison and second-order reread;
- the analysis could fit an unrelated chapter unchanged.

The receipt is not invalid merely because:

- a meaningful witness remains silent;
- an optional texture line is removable;
- a background character has no line;
- a natural competence has no trigger in this scene;
- the draft uses equivalent wording or different minor staging.
